A pasta data armazena a seguinte estrutura de pastas:
- Input (armazena arquivos de input para o processo)
- Output (armazena arquivos de saída do processo)
- Tempo (armazena arquivos que serão utilizados ao longo do processo de forma temporária)

Armazena também os seguintes arquivos:
- config.json (armazena variáveis de configuração do projeto)
- applications.json (armazena variáveis de inicialização/encerramento de aplicações utilizadas no processo)
